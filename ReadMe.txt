Software Details:

1. Install Eclipse Luna to import the project.

2. Download Maven from Install software in Eclipse.

3. Download Cucumber plugin from Install software.

4. Download TestNG Framework.

5. In the Pom.xml, required Cucucmber, Selenium, TestNG and other jar files have been placed.

6. To execute the Test, do - Run as --> maven clean install.(inorder to compile, change the properties of TestNG in pom.xml to Compile)

7. Java 1.8 is the JVM used as execution environment.
